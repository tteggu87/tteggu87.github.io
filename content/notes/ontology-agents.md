# 온톨로지 에이전트 시작점

온톨로지 에이전트는 단순히 그래프를 검색하는 에이전트보다 범위가 넓습니다. 클래스와 속성의 의미, 제약, 버전이 실제 계획과 검증 경로에 참여해야 합니다.

## 안전한 기본 흐름

```text
근거 수집 → 변경 후보 → 격리된 제안 그래프 → 규칙 검증 → 사람 승인 → 버전 릴리스
```

LLM은 근거가 붙은 후보를 만들고, 결정적 검사와 사람이 최종 지식 구조를 확정하는 방식이 방어적입니다. 실행 주체의 설계는 [[notes/ai-agents|AI 에이전트 작업노트]], 근거 관리 방법은 [[notes/research-systems|리서치 시스템 설계]]와 연결됩니다.
